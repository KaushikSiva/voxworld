# HumanVoice Demo Script

## 60-second version

1. Open the app and point to the verification badge.
2. Click `Connect Coinbase Wallet`.
3. Say: `This agent now has a real wallet identity, but that alone is not enough for protected inventory.`
4. Leave it unverified and press the mic button.
5. Say: `Book one coworking pass for tomorrow.`
6. Say: `The assistant understands the request, but the merchant rejects it because protected inventory requires a verified human-backed agent.`
7. Point to the event log and rejection card.
8. Click `Verify with World ID`.
9. Repeat: `Book one coworking pass for tomorrow.`
10. Say: `Now the same wallet-backed agent is also backed by verified human proof through World ID, so the merchant approves the action.`
11. Let the spoken success response play.

## One-line pitch

HumanVoice lets voice agents act on protected inventory only when a wallet-backed agent can also prove it is backed by a real unique human.

## Suggested backup prompts

- `Reserve one event ticket for me.`
- `Buy one coffee subscription.`

## Key judging points

- Same UI, same agent, same request
- Unverified path fails
- Verified path succeeds
- Merchant trust gate is visible and understandable
